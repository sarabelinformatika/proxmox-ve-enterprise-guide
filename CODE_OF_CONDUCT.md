# Code of Conduct

## Our Commitment

This project aims to provide high-quality educational resources for IT professionals.

We are committed to maintaining a welcoming, respectful, and professional environment for everyone regardless of experience, background, or organization.

---

## Expected Behavior

Participants are expected to:

- Be respectful and constructive.
- Provide professional and technical feedback.
- Help improve documentation accuracy.
- Focus discussions on technical topics.
- Respect differing opinions.

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Personal attacks.
- Harassment.
- Discrimination.
- Offensive language.
- Spam or promotional content unrelated to the project.
- Deliberate misinformation.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation

---

## Enforcement

Project maintainers reserve the right to remove comments, contributions, or participants that violate these guidelines.
