# Security Policy

## Supported Versions

Security improvements are applied only to the latest published version of this documentation.

Please use the most recent release whenever possible.

---

## Reporting Documentation Issues

If you discover:

- incorrect security recommendations;
- outdated security guidance;
- inaccurate technical information;
- potentially unsafe documentation;

please open a GitHub Issue.

If the issue could negatively impact production environments, please describe it responsibly and avoid publishing sensitive details.

---

## Scope

This repository contains documentation only.

It does not distribute executable software or configuration packages.

Security reports should therefore relate to:

- documentation accuracy;
- security recommendations;
- enterprise best practices.
